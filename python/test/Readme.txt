1. build proto to python code. 'protoc' is generated by protobuf
> protoc -I ../idl --python_out=. ../idl/common.base.Example.proto

2. run client/server.
    PYTHONPATH contains:
        a) path to python version of protobuf;
        b) path to generated code 
        c) path to fdbus.py
    LD_LIBRARY_PATH contains path to libfdbus-clib.so
> PYTHONPATH=path_to_protobuf_python:./common/base:.. LD_LIBRARY_PATH=path_to_libfdbus-clib.so:$LD_LIBRARY_PATH python fdbus_test_client.py server_name
> PYTHONPATH=path_to_protobuf_python:./common/base:.. LD_LIBRARY_PATH=path_to_libfdbus-clib.so:$LD_LIBRARY_PATH python fdbus_test_server.py server_name
